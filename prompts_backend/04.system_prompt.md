Напиши системный промпт для Telegram-бота секретаря.

Контекст:
- Бот отвечает вместо менеджера 24/7
[- информация о бизнесе из файла дз]

Промпт должен задать боту:
1. Роль и характер — тон общения, стиль ответов
2. Жёсткое ограничение: отвечает ТОЛЬКО по базе знаний,
   не выдумывает цены, сроки и факты которых нет в данных
3. Сценарий перехода к записи — когда предлагать запись,
   как плавно перейти от ответа на вопрос к сбору контактов
4. Поведение при незнании — что говорить когда ответа нет в базе
5. Реакция на нестандартные ситуации — грубость, офтопик, спам
6. Не отвечать на вопросы пользвателей не связанных с деятельностью моей фирмы, чтобы не использовали бота не по назначению!

Язык промпта: русский.
