Напиши модуль rag — pipeline для поиска по базе знаний.

Функции:
- load_knowledge(filepath) — читает knowledge.md, 
  делит на чанки по разделителям, генерирует embeddings

- search(query, top_k=3) — принимает вопрос пользователя,
  генерирует его embedding, ищет top_k похожих чанков 
  через оператор pgvector <-> (cosine distance),
  возвращает список текстов
