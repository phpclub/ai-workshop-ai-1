Спроектируй структуру Telegram-бота секретаря с RAG. берем стек из 07-stack-research.md

Функции:
- Отвечает на вопросы по базе знаний компании
- Собирает заявки: имя → телефон → вопрос → уведомляет владельца в Telegram
- База знаний редактируется через текстовый файл без программирования
- Отвечает на вопросы по теме, чтобы мы не тратили ресурсы API

Покажи:
1. Дерево файлов проекта с объяснением каждого файла
2. За что отвечает каждый модуль (одно предложение)
3. Какие переменные окружения нужны в .env и почему каждая там

Принцип: один файл — одна ответственность.
Объясни каждое архитектурное решение.
